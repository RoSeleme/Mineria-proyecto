#Proyecto Final - Minería de Datos
Dashboard interactivo de siniestros viales fatales en Argentina (2017–2023).
El proyecto utiliza Streamlit para visualización y pandas para análisis.

Este repositorio contiene:

siniestros_limpio.csv (dataset procesado)
app.py (código del dashboard)
requirements.txt (dependencias de la aplicación)
El dashboard se despliega en Streamlit Cloud.
